HEXO 常用命令

May 22, 2018

初始化工程

  • hexo init MyBlog

本地预览

  • 安装hexo server：sudo npm install hexo-server
  • 生成静态页面：hexo g
  • 打开hexo本地服务(指定端口5000，默认4000)：hexo server -p 5000

git部署

  • 安装hexo的git部署：npm install hexo-deployer-git --save
  • 生成静态页面并部署到github：hexo d -g（或hexo g+hexo deploy）
